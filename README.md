# Sistema de Controle de Hábitos

## Sobre o projeto

Este é um **projeto simples de terminal** feito em **Python** que utiliza **PostgreSQL** como banco de dados para registrar e acompanhar hábitos.

O objetivo do projeto é **praticar integração entre programação e banco de dados**, utilizando ferramentas comuns no desenvolvimento backend.

## Tecnologias utilizadas

* Python
* PostgreSQL
* psycopg2

## Funcionalidades atuais

* Criar hábitos
* Marcar frequência de hábitos
* Deletar hábitos
* Visualizar histórico de registros
* Resetar o banco de dados

## Objetivo do projeto

Este projeto foi criado com fins de **aprendizado**, explorando conceitos básicos como:

* conexão com banco de dados
* criação de tabelas
* operações CRUD
* manipulação de dados via terminal

## Planos futuros

Algumas melhorias planejadas:

* Melhor organização do código
* Estruturação em funções
* Melhor tratamento de erros
* Interface web para interação com os hábitos
* Visualização mais clara de estatísticas

## Como executar

1. Instale as dependências:

```
pip install psycopg2
```

2. Certifique-se de ter o **PostgreSQL** instalado e rodando.

3. Configure suas credenciais no código.

4. Execute o programa:

```
python main.py
```
